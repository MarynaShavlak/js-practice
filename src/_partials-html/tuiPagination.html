• Tui Pagination

    Завантаження додаткових зображень зображено за допомогою блоку із номерами сторінок

    Сервіс зображень: unsplash

    Сервіс для відображення повідомлень користувачу: notiflix

    Спіннер для відображення процесу очікування результатів завантаження реалізовано за допомогою spin.js

    Для HTTP-запитів використана бібліотека : axios

    Для створення блоку пагінації використовується бібліотека tui pagination
