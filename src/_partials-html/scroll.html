• Event "Scroll"

    Коли користувач скролить сторінку і прокручує її на 800 і більше пікселів, то анімовано з'являється секція номер 4

    Для визначення значення положенгя вертикальної прокрутки використовується властивість Window.pageYOffset

    Для генерування випадкового кольору у hex-форматі використовується функція randomColor()

    const randomColor = () => '#' + (Math.random().toString(16) + '000000') .substring(2, 8).toUpperCase();

    Для обробки скролу використовується бібліотека lodash
