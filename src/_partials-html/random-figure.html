• Random figure change

    Спочатку на екрані користувача відображається фігура рандомної форми. При натисканні на неї в рандомному порядку форма фігури змінюється на іншу.Фігура щоразу з'являється у різних місцях на сторінці. Колір фігури в рандомному порядку змінюється.

    Для генерування випадкового кольору у hex-форматі використовується функція randomColor()

    const randomColor = () => '#' + (Math.random().toString(16) + '000000') .substring(2, 8).toUpperCase();

    Для генерування випадкового числа від 0 до max (НЕ включно)

    function randomither(max) { return Math.floor(Math.random() * max); }
