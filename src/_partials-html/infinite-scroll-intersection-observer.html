• Infinite scrol with "Intersection Observer API"

    Пошук новин за запитом користувача та відображення галереї за результатами запиту. Завантаження додаткових новин реалізовано за допомогою "нескінченного скролу".

    Для створення розмітки використано шаблонізатор handlebarsjs.com

    Сервіс новин: newsapi.org

    Нескінченний скрол реалізовано за допомогою : Intersection Observer API
