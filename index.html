• Use LocalStorage- Форма для заповнення данних

    Запам'ятати дані, які користувач вводить в поля форми, зберегти у локальному сховищі. При перезавантаженні сторінки (її наступному відвідуванні) відновити дані, тобто поля форми заповнити тим, що користувач вводив до цього.

  • Use LocalStorage - Перемикач теми сайту

    Користувач може обрати в інтерфейсі світлу чи темну тему сайту, при перезавантаженні сторінки чи наступному її відвідуванні сайту обране налаштування теми збережеться

  • LocalStorage Contact Form- Форма для заповнення данних

  • Notification - Самозникаюче сповіщення

    Після проведення операції для користувача з'являється сповіщення про її успішність. Це сповіщення має зникате у двох випадках: 1. Користувач натискає на сповіщення. 2. Сповіщення зникає через 5 секунд після того як з'явилось

  • Annoying Subscription

    Користувачеві з'являється "набридаюче" вікно із пропозицією підписатись. Якщо користувач натискає на "закрити", то вікно закривається, але через 5 секунд з'являється знову. Максимум повідомленя може вспливати 3 рази. Якщо користувач натиснув на кнопку "підписатись", то набридаюче вікно більше не з'являється

  • StopWatch

    При натисканні на кнопку "запустить" починається робота секундоміру. Натиск на кнопку "остановить" зупиняє секундомір та скидає значення до нуля

  • Countdown Timer
